Read Me

In this puzzle game I tried to handle game functions from both front-end and back-end and front-end only. You can choose that option from
environment.ts -> connectedWithAPI: true / false.

To run this project first open montyhall project main folder and navigate using 'cd montyhall' to attach with the project.

Set number of rounds before start the game.Then you can play the game until given rounds completed.This puzzle game is developed to reach the required scope as much as possible and with extra things like page navigations.
